By default, dodging with `position_dodge2()` preserves the total width of the elements. You can choose to preserve the width of each element:


<pre class="mcode">
p <-    
 ggplot(mtcars, aes(factor(cyl), fill = factor(vs))) +
  geom_bar(position = position_dodge2(preserve = "single"))
  
plotly::ggplotly(p)
</pre>

# Dodge error bars

  geom_col(position = "dodge", fill = "grey50", colour = "black")
