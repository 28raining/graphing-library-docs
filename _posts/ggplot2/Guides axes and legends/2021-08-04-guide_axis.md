# Overlapping text in x-axis

Axis guides can be customized in the `scale_*` functions or using `guides()`


<pre class="mcode">
p <- ggplot(mpg, aes(cty * 100, hwy * 100)) +
  geom_point() +
  facet_wrap(vars(class))

p <-  p + scale_x_continuous(guide = guide_axis(n.dodge = 2))

plotly::ggplotly(p)
</pre>
