--- name: Plot A Covariance Ellipse plot_url: https://plotly.com/~tarzzz/2062.embed arrangement: horizontal language: matlab suite: basic_shapes order: 2 sitemap: false description: Inspired from Stack Overflow --- num = 50; X = [ mvnrnd([0.5 1.5], [0.025 0.03 ; 0.03 0.16], num) ; ... mvnrnd([1 1], [0.09 -0.01 ; -0.01 0.08], num) ]; G = [1*ones(num,1) ; 2*ones(num,1)]; fig = figure; gscatter(X(:,1), X(:,2), G) axis equal, hold on for k=1:2 idx = ( G == k ); Mu = mean( X(idx,:) ); X0 = bsxfun(@minus, X(idx,:), Mu); [V D] = eig( X0'*X0 ./ (sum(idx)-1) ); [D order] = sort(diag(D), 'descend'); D = diag(D); V = V(:, order); t = linspace(0,2*pi,100); e = [cos(t) ; sin(t)]; VV = V*sqrt(D); e = bsxfun(@plus, VV*e, Mu'); plot(e(1,:), e(2,:), 'Color','k'); end fig2plotly(fig)
