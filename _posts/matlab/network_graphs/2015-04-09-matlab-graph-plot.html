Matrices and MATLAB's gplot function plot_url: https://plotly.com/~PlotBot/109 arrangement: horizontal language: matlab suite: network_graphs order: 0 sitemap: false --- % Create the onnectivity graph of the Buckminster Fuller geodesic dome [B, V] = bucky; H = sparse(60, 60); k = 31:60; H(k, k) = B(k, k); % Visualize the graph using the gplot function (blue) fig = figure; gplot(B - H, V, 'b-'); hold on; % Visualize a rotation of the graph (red) gplot(H, V, 'r-'); axis off equal; %--PLOTLY--% % Strip MATLAB® style by default! fig2plotly(fig);
