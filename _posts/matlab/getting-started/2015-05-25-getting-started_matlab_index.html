Installation

Download and uncompress the Plotly Graphing Library for MATLAB®.

Initialization

In your MATLAB® console, begin by locating the downloaded directory, and running plotlysetup_offline(). 

>> cd ~/Downloads/plotly-graphing-library-for-matlab-master
>> plotlysetup_offline()

Or, if you have a plotly bundle url like 'http://cdn.plot.ly/plotly-latest.min.js' 

>> cd ~/Downloads/plotly-graphing-library-for-matlab-master
>> plotlysetup_offline('plotly_bundle_url')

You'll need to replace "plotly_bundle_url" with with a valid Plotly bundle url.

Start Plotting Offline!

Now you can convert your MATLAB® figures into beautiful, interactive, Plotly graphs offline with fig2plotly(). The graphs will be saved as html files to your current working directory and contain all of the necessary dependencies to render the plot within a browser offline. 



[X,Y,Z] = peaks;
contour(X,Y,Z,20);

fig2plotly()

Please see Getting started with Chart Studio to sync your MATLAB plots with your free online Chart Studio account.
